Real-time twitter streaming and sentiment analysis


Goal of this project is to connect to Twitter API and continuously stream tweets of a specific location or keyword and then to perform real-time processing on the streamed tweets on the fly to calculate sentiment analysis based on the tweet text, and eventually visualize the results real-time.


Tools & Technologies


Twitter API, Kafka, Spark, Flask, HTML, CSS, Bootstrap, Chart.js, Leaflet.js, Mapbox API


How it works?


Go to 'Get Started' and enter any single keyword, select a location from the drop-down and hit Submit. Now, you can go to Trending to check the top ten trending hashtags on twitter based on the location entered, Map to check the real-time sentiments being calculated on the tweets received from the entered location and Keyword page to check the sentiment of the tweets which contains the keyword entered
